# Flipfolio

A modern NFT trade journal and gallery built with Next.js, TypeScript, Tailwind CSS, and Zustand.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Features

- Add, edit, and delete NFT trade records
- Persist records to LocalStorage via Zustand
- Filter by owned or sold status
- Search by NFT name or collection
- Auto-calculate profit/loss, ROI, and holding duration
- Dashboard summary cards with totals, win rate, and average hold time
- Seeded demo data for first launch
